Java编写单词小游戏

发表于 2020-08-03 阅读次数：

介绍

公司最近有一个竞技场项目，里面有一个单词小游戏。

游戏大概就是随机生成一个5*5的棋盘，上面有单词的字母，通过滑动连出正确的单词。

棋盘生成算法

思路

首先随机选个一个起点，从这个点开始铺单词。

分别选取上下左右四个方向作为下一个字母的摆放位置，不能触边也不能走重复路，直到平铺完所有的单词。

如果在棋盘能平铺下单词的情况下，找不到路径，就从四个角作为起点，必能找到路径。
